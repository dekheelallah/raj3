# ملاحظات الإطلاق — راجع

## رابط الموقع المنشور

https://raj3-six.vercel.app/

## حالة النسخة الحالية

هذه نسخة MVP أولية منشورة على Vercel.

## ما يعمل حاليًا

- واجهة عربية RTL
- صفحات الموقع الأساسية
- إنشاء طلب توصيل تجريبي
- حفظ الطلب مؤقتًا داخل المتصفح باستخدام localStorage
- تتبع الطلب التجريبي من نفس المتصفح
- التحقق من صيغة رقم الهاتف الكويتي +965
- التحقق من صيغة رقم الطلب RJ-123456

## ما لا يعمل بعد

- لا توجد قاعدة بيانات حقيقية
- لا يوجد تسجيل دخول فعلي
- لا يوجد تحقق OTP
- لا يوجد دفع إلكتروني
- لا توجد لوحة إدارة
- لا يوجد Backend API
- لا يوجد تطبيق iOS

## ملاحظة مهمة

الموقع المنشور ليس جاهزًا تجاريًا بعد.  
أي طلب يتم إنشاؤه محفوظ داخل متصفح المستخدم فقط، ولا يصل إلى خادم أو قاعدة بيانات.
