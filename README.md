# راجع

منصة ويب أولية لخدمة توصيل داخل الكويت فقط.

## التقنية المستخدمة

- React
- Vite
- TypeScript
- Tailwind CSS
- React Router
- RTL Arabic UI
- Mobile-first design

## الصفحات الحالية

- الصفحة الرئيسية
- اختيار نوع المستخدم
- تسجيل الدخول
- إنشاء حساب
- طلب توصيل
- تتبع الطلب
- الأسعار
- الدعم
- سياسة الخصوصية
- الشروط والأحكام
- حالة المشروع
- صفحة 404

## القيود الحالية

هذا المشروع حاليًا نسخة أولية MVP فقط.

لا يحتوي بعد على:

- قاعدة بيانات حقيقية
- تسجيل دخول فعلي
- نظام دفع
- لوحة إدارة
- API Backend
- حماية إنتاجية
- تطبيق iOS

## التخزين الحالي

يتم حفظ الطلبات مؤقتًا داخل localStorage في المتصفح فقط.

هذا التخزين مناسب للاختبار، وليس مناسبًا للإطلاق التجاري أو حفظ بيانات العملاء الحقيقية.
